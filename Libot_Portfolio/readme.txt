Madelyn Eway Portfolio Website

Description:
This is my personal portfolio website, showcasing my projects, skills, talents, organizational involvement, services, testimonials, and contact information.

Features:
- Responsive sidebar navigation
- Home section with profile and social links
- Portfolio section with project listings
- Skills & Talents section
- Organizations section
- Services section (populated from XML)
- Testimonials section (populated by JavaScript)
- Contact section with contact info and a demo form

Technologies Used:
- HTML5, CSS3
- JavaScript (for dynamic content)
- Font Awesome for icons
- Google Fonts (Poppins, Montserrat)

Instructions:
1. Place all static files (CSS, JS, images) in the `/static` directory.
2. Edit `index.html` to update content as needed.
3. To add testimonials, projects, skills, talents, or organizations, update the corresponding JavaScript or XML data sources.
4. The contact form is for demo purposes only and does not send real emails.

Contact:
For questions or collaboration, reach out via the contact section on the website.
